# Reference Resources

This folder keeps project reference material separate from the runnable prototype.

- `SPEC.md`: product direction for the AI-based Caption with Intention creator.
- `CAPTION_WITH_INTENTION_INSTRUCTIONS_AND_GUIDELINES.md`: implementation-oriented CWI guidelines synthesized from the source PDFs.
- `Caption with Intention – Quickstart Guide.pdf`: original quickstart source document.
- `Caption-With-Intention_Design-System_V1.0.pdf`: original design-system source document.
- `Instal Font - RobotoFlex.ttf`: local Roboto Flex font used by the prototype caption renderer.
- `AE PROJECT/`: After Effects template assets and bundled sample video used by the default local preview.

The root demo still loads the sample video from `reference/AE PROJECT/(Footage)/ASETS/Video/Cena_ref_CI_Template_v02a.mp4`.
