# Source Layout

- `styles.css`: visual layout, responsive behavior, and component styling.
- `model.ts`: CWI constants, speaker palettes, sizing constants, and seed project data.
- `app.ts`: DOM wiring, rendering, editor interactions, import/export, and playback sync.

Run `npm run build` after editing TypeScript. `index.html` loads the compiled `dist/app.js` so the prototype still works from `file://`.
